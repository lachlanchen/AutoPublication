> Автоматическая публикация видео с помощью инструментов ИИ.

## 🧭 Обзор

`AutoPublication` — это каркас, ориентированный на документацию, предназначенный для определения рабочих процессов публикации видео с помощью ИИ.

На текущем этапе репозитория проект остаётся каркасом с документацией и артефактами пайплайна генерации README, но исходный код приложения или точка входа в рантайме ещё не внесены.

Этот README является каноническим источником английской документации и структурирован так, чтобы поддерживать будущий рост внедрения без потери текущего замысла проекта.

## ✨ Особенности

### ✅ Текущие возможности

- Каноничный корневой README для определения проекта.
- Предопределённые цели многоязычных README в `i18n/`.
- Артефакты контекста и структуры README-пайплайна в `.auto-readme-work/`.
- Строка языковой навигации централизована вверху для мультиязычного соответствия.

### Планируемые возможности (выводимые из названия проекта и текущего описания)

- Автоматизированная оркестрация публикации видео.
- Подготовка метаданных и контента с поддержкой ИИ.
- Настраиваемые назначения публикации и интеграции с платформами.

## 🩺 Устранение неполадок

### «Не могу запустить проект»

Причина:

- Исходные файлы приложения или текущий entrypoint ещё не зафиксированы.

Решение:

- Рассматривайте репозиторий как каркас/документационную основу до добавления реализации.

### «Ссылки на локализованные README есть, но файлов нет»

Причина:

- `i18n/` существует, но локализованные README-файлы не сгенерированы в этом снимке.

Решение:

- Сгенерируйте и добавьте файлы, перечисленные в `i18n/README.*.md`, и артефакты плана перевода.
